# Estilo Cool – Tienda de Ropa con Panel Admin

Tienda virtual con carrito de compras, pedidos por WhatsApp y panel de administración para gestionar productos.

## Características

- Diseño moderno.
- Catálogo con imágenes, precios, talles y stock.
- Carrito lateral persistente (localStorage).
- Subir fotos propias desde la tienda o desde el admin.
- Panel de administración con contraseña para:
  - Editar nombre, precio, stock, talles.
  - Cambiar imagen.
  - Eliminar productos.
  - Agregar nuevos productos.
- Pedido automático por WhatsApp.

## Tecnologías

HTML, CSS, JavaScript (ES6), localStorage.

## Instalación

Solo descarga los 3 archivos (index.html, styles.css, app.js) y ábrelos en tu navegador.

## Personalización

- Número de WhatsApp: en app.js, variable `phoneNumber`.
- Contraseña admin: en app.js, línea `if (pass'')`.
- Colores: en styles.css, variables `:root`.
